Documentação da Fase 3 - Simulação de Sistema de Irrigação

Este projeto simula uma máquina agrícola inteligente com ESP32, sensores e controle de bomba de irrigação.

Componentes:
- ESP32
- Sensor de umidade DHT22
- Sensor de pH simulado com LDR
- Botões para fósforo e potássio
- Relé como bomba de irrigação

Entregas:
- Código fonte C++ (PlatformIO)
- Circuito simulado no Wokwi
- Script Python com SQLite e CRUD
- Dashboard (opcional)
- API Clima (opcional)
